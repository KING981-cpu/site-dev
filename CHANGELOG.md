# Changelog

Todas as mudancas relevantes deste projeto serao documentadas aqui.

## v1.0.0 - 2026-03-09

- refatoracao completa da interface para `app.html` com arquitetura modular
- remocao da dependencia obrigatoria de backend no runtime principal
- persistencia local-first com backup manual, import/export JSON e auto-backup
- observabilidade local com audit log, captura de erros, boot time e footprint
- pagina operacional `operacao.html`
- documentacao profissional com README, release notes e workflows GitHub Actions
- smoke test browser headless e testes unitarios/estruturais
