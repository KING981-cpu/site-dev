# Release v1.0.0

Data: 2026-03-09

## Objetivo

Transformar a base original em um workspace profissional, testavel e operavel sem backend obrigatorio.

## Entregas

- nova aplicacao principal em `app.html`
- UI responsiva e modular com `assets/styles/app.css`
- dominio separado em `assets/scripts/core.js`
- runtime de interface em `assets/scripts/app.js`
- backup manual, import/export JSON e restore
- auto-backup a cada 5 mutacoes de dados
- observabilidade local com audit log e captura de falhas
- smoke test browser real
- CI, Pages e workflow de release

## Riscos mitigados

- dependencia de CDN/backend para fluxo critico
- logica concentrada em um unico HTML monolitico
- ausencia de backup e restore
- falta de validacao automatizada
- falta de documentacao de release

## Validacao executada

- testes unitarios do nucleo
- testes estruturais de arquivos e rotas
- smoke test browser headless cobrindo:
  - navegacao
  - toggle de tema
  - tarefas
  - notas
  - atalhos
  - calendario
  - focus timer
  - backup e restore
  - exportacao
  - integridade da observabilidade
